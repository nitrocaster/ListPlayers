1.0 (12.08.2010)
   Первая релизная версия.

1.1 (26.09.2010)
 * Добавлена возможность сбора данных из конфиг-дампов и скриншотов;
 * Исправлен неполный сбор данных из дампов статистики, записываемых после конца матча;
 * Доработан алгоритм обработки файлов, в результате чего повысилось быстродействие и уменьшился объем занимаемой памяти;
 * Исправлена ошибка, из-за которой в базу могли записываться пустые ники, ip-адреса и идентификаторы профилей;

1.2 (24.11.2010)
 * Добавлена возможность объединения баз.

2.0 Beta (29.11.2011)
 * Изменен формат баз данных, использована встраиваемая СУБД SQLite;
 * Изменения в структуре БД: добавлены комментарии к каждому хэш-коду и отметки времени для имен, IP и GameSpy ID;
 * Реализован поиск по базе данных с экспортом найденного в текстовый файл;
 * Добавлено сохранение путей и состояний элементов управления.

2.0.1 Beta (01.12.2011)
 * Устранена несовместимость с Windows XP;
 * Исправлен вылет при обработке дампа статистики, не содержащего секций игроков.

2.0.4 (03.03.2012)
 * Исправлен вылет при обработке баз данных старого формата с ошибками;
 * Исправлен вылет при обработке ников игроков, содержащих специальные символы;
 * Двухкратный прирост скорости вывода результатов поиска;
 * Добавлен экспорт в формат RADB (для RAdmin Panel).
 
2.0.5 (25.03.2012)
 * Исправлено отображение элементов интерфейса в Windows XP;
 * Добавлена фильтрация пустых ip-адресов и ников при обновлении базы из дампов статистики, конфиг-дампов и скриншотов;
 * Добавлена возможность поиска по шаблону хэш-кода;
 * Добавлено обновление отметок времени в базах.
 
2.0.6 (04.09.2012)
 * Исправлена ошибка, из-за которой в окне обновления базы показывалось неверная статистика добавленных записей;
 * Исправлена ошибка, из-за которой в базу вносились неверные отметки времени;
 * Исправлен порядок расположения записей в истории путей;
 * При обновлении из старой базы данных выставляются отметки времени DateTime.MinValue;
 * Пустые хэш-коды и соответствующие им данные игнорируются (но попадают в статистику найденных);
 * Добавлена автоматическая проверка обновлений.
 
2.0.6.1 (04.09.2012)
 * Исправлена ошибка, связанная с неверным форматом даты при обработке конфиг-дампов и скриншотов.

2.0.6.2 (11.04.2013)
 * Обновлена используемая библиотека SQLite;
 * Добавлена проверка версии игры при обновлении из старой текстовой базы;
 * Добавлено обновление из SQLite баз (теперь новые базы можно объединять).
